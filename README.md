# Finora

Smart Monthly Planning For Modern Indian Families

## Features

- Monthly salary planning
- Investment allocation
- EMI & loan health tracking
- Emergency safety planning
- Family financial goal planning
- Responsive fintech UI
- Mobile-first design

## Run Locally

Install dependencies:

npm install

Start development server:

npm run dev

Open:
http://localhost:3000

## Deployment

This project is configured for Netlify deployment.

Build command:
npm run build

Publish directory:
Leave empty in Netlify settings.
